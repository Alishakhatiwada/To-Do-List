# React To-Do App

A scalable To-Do application built with React, React Router, Tailwind CSS, and LocalStorage.

## Features
- Add, Edit, Delete tasks
- Mark tasks as Completed/Pending
- Filter tasks by status
- Search tasks by title
- Data persistence using localStorage

## Run Locally
-cd To-Do
-npm run dev

## Folder Structure
src/components
src/pages
src/utils
